# Calculadora de Ativos Fixos

## Instalação

Crie um virtual environment:

```bash
>> python3 -m venv .venv
```

Ative o novo virtual environment:

```bash
>> source .venv/bin/activate
```

Instale as dependências:

```bash
>> pip install -r requirements.txt
```

## Desenvolvimento

```bash
>> flask run --debug
```

## Produção

```bash
>> flask run
```
